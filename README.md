# Apple Clone

This is an Apple website clone that I built while learning HTML, CSS, and JavaScript.
The main goal of this project was to improve my frontend development skills by recreating the design and layout of a real-world website.

## Features

 Responsive navigation bar
 Product sections inspired by Apple's website
 Clean and modern UI
 Responsive layout
 Basic JavaScript interactions

## Technologies Used

* HTML
* CSS
* JavaScript

## What I Learned

While building this project, I practiced:

 HTML page structure
 CSS styling and layouts
 Flexbox
 Responsive design
 JavaScript basics
 Website cloning and UI development

## About This Project

This project is part of my web development learning journey. I created it to gain hands-on experience and better understand how professional websites are designed and built.
